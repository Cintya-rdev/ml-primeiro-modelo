# Meu Primeiro Modelo de Machine Learning

## Descrição
Projeto de aprendizado de Machine Learning desenvolvido durante a graduação em Engenharia Química na UFMT.

## Objetivos
- Aprender os fundamentos de ML com Python
- Aplicar modelos de classificação e regressão
- Documentar o processo de aprendizado

## Tecnologias
- Python 3.14
- Scikit-learn
- Pandas
- Matplotlib

## Estrutura do Projeto
- `data/` → dados utilizados no projeto
- `notebooks/` → experimentos e análises
- `src/` → código Python organizado
- `outputs/` → gráficos e resultados gerados
